## Description

### _This project is a modern, interactive e-commerce web application that integrates a voice assistant to create a seamless and user-friendly shopping experience. The application is designed to allow users to interact with the platform using voice commands, making it accessible and efficient. This unique functionality simplifies common actions like browsing products, searching for specific items by category or name, and managing a shopping cart, all without requiring manual input._

_The voice assistant leverages advanced speech recognition and text-to-speech technology to enhance usability. Users can speak commands to search for products, scroll through the page, or interact with the cart, and the application provides real-time audio feedback. This makes the shopping process not only more engaging but also accessible to users with physical disabilities or those who prefer hands-free interaction._

_Built with a combination of modern web technologies like HTML, CSS, JavaScript, and Bootstrap, the platform features a responsive and visually appealing design. It is optimized for both desktop and mobile devices, ensuring that users enjoy a consistent experience across different platforms. The integration of the FakeStoreAPI provides dynamic product data, giving the application the feel of a live e-commerce store._

_The shopping cart functionality is a central feature of the application. Users can add items to their cart, adjust quantities, and view a summary of their selected products. The cart state is preserved using localStorage, ensuring that users can return later and resume their shopping seamlessly. The voice assistant also includes commands for managing the cart, such as checking the number of items or toggling the cart view._

### Key Features:
* Voice Interaction: Perform key shopping actions with simple voice commands.
* Dynamic Product Browsing: Browse and filter products by category or search for specific items.
* Responsive Navigation: Voice commands allow users to scroll through pages or jump to the top or bottom.
* Cart Management: Add, remove, and view items in the cart with real-time updates.
* Accessible Design: Hands-free controls and intuitive UI make the platform suitable for a wide range of users.
* Auto Scroll: Gradual automatic scrolling provides a unique hands-free browsing experience.
* Interactive Feedback: Real-time voice feedback confirms user actions, creating a conversational experience.
* This project represents the future of online shopping by combining traditional e-commerce features with the power of voice technology. It is not just a demonstration of modern web development techniques but also a step towards making technology more inclusive and intuitive for all users.

### Usage Instructions
* üöÄ Voice Commands
* Search by Category:
* Say: "Category Electronics"
* Search by Product:
* Say: "Product Laptop"
* Scroll:
* "Scroll down": Scrolls one screen down.

* "Click C": Toggles the cart view.
